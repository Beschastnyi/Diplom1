# Diplom1
Это приложение предназначно для людей, которые следять за тем, что они кушают.

Приложение предлагает пользователм верифицироваться через электронную почту(для этого нужно перейти
в Registration Activity нажав на кнопку register в начальном окне)

В нем содержатся два основных Activity(1 - Profile: в котором отображаются данные пользователя;
2 - ListOfEating: в котором можно содержится список, того что съел пользователь, при нажатии
на элемент списка, пользователь может увидить количество полученных белков, жиров и углеводов).

List of eating  также содержит в себе кнопку добавления нового блюда, при нажатии на нее пользователь
переключатся на ChooseEating Activity, где он выбирает блюдо, которое он желает съесть.

После выбора блюда открывается AddEating Activity, в котором указанна питательная ценность выбранного
блюда на сто грамм. Там он вводит количество в граммах, сколько он съел выбранного блюда. После нажатия
кнопки сохранения результат сохраняеться и как элемент списка появляется в ListOfEating Activity.

ListOfEating Activity позваляет пользователю удалить элемент из списка после его подтверждения.
В Profile Activity есть возможност с галереи добавить свою картинку при нажатии на картинку.
